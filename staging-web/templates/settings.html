Setup Instructions
GitHub Token Setup
  1. Go to GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Generate new token
  3. Select "kernelci/kernelci-core" repository
  4. Grant "Actions" permissions (write)
  5. Copy token and paste above
Discord Webhook Setup
  1. Go to your Discord server settings
  2. Navigate to "Integrations" → "Webhooks"
  3. Click "Create Webhook"
  4. Choose a channel and customize the webhook name/avatar
  5. Copy the webhook URL and paste above

  • Discord: Real-time notifications for staging run events
  • Self-Update: Optional git pull step to update staging scripts
  • Kernel Trees: Automatic rotation between next, mainline, and stable
  • Docker: Automated API and pipeline deployment
